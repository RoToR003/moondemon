is a simple web-based tool with the ability to scan diagnostic sweats and stress tests


## Installation

Install moondemon with git

```bash
apt update
pkg upgrade
pkg install git
pkg install python
git clone https://github.com/RoToR003/moondemon.git
```
    
## Usage/Termux

```bash
cd moondemon
pip install -r requirements.txt
python main.py
```
